# DashBoard

## Install dependencies

```bash
npm install
```

## Add .env

```
# environment
NODE_ENV="development"

# Auth.js secret
AUTH_SECRET="your-atleast-32-characters-long-secret"

# Google client OAuth credentials
GOOGLE_CLIENT_ID="your-client-id"
GOOGLE_CLIENT_SECRET="your-client-secret"
```

## Run

```bash
npm run dev
```
